# Computer Vision
Hi

Thanks for visiting my Computer Vision respository.

In this repository's individual files, I use
1. CIFAR image dataset to classify 10 different everyday objects with Neural network (NN)
2. CIFAR image dataset to classify 10 different everyday objects with Convolutional neural network (CNN)
3. a dataset to classify cats and dogs with CNN
4. VGG16 model to classify cats and dogs with dropout & imagedatagenerator
5. VGG16 model to classify a dog and a piano
6. MobileNetV2 model to classify cats and dogs
7. fashion_mnist dataset to classify types of clothes with NN
8. mnist dataset to classify handwritten numbers with NN
9. mnist dataset to classify handwritten numbers with CNN

In the folders, I used OpenCV to do 3 capstones: 
1. deep learning vision model to detect simpsons faces
2. face detection: haar_face cascades
3. face recognition: build a recognizer based on features, and labels, save model as a YAML, read the saved model, and make predictions on an image.

I also learned how to use openCV to: 
1. reading images & video,
2. resizing and rescaling frames
3. drawing shapes and putting text
4. 5 essential functions: converting an image to grayscale, blurring, edge cascade, dilating, eroding the dilated image, resizing, and cropping using array slicing
5. Image transformations: translation, rotation, resizing, flipping, and cropping
6. contour detection: canny edge detector, and binarize
7. color spaces: BGR to grayscale, BGR to HSV, BGR to RGB, grayscale to BGR, BGR to LAB, and HSV to BGR
8. color channels: split an image into 3 respective color channels, reconstruct an image to display the actual color, merge those color channels back into its original image
9. blurring: averaging, gaussian, median, bilateral
10. bitwise: AND, OR, XOR, NOT (e.g., with rectangle and circles)
11. masking (focus on certain parts of an image e.g. focus on faces and remove everything else)
12. histogram computation: analyze the distribution of pixel intensities for grayscale, or colored images
13. Thresholding/binarizing: simple, and adaptive
14. edge detection: laplacian, sobel, canny

This tutorial is from the freecodecamp & 2 day courses.
